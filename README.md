## Langkah-langkah Menjalankan Aplikasi

1. Jalankan migrasi database dengan perintah:
php artisan migrate

3. Isi database dengan data awal menggunakan perintah:
php artisan db:seed

3. Untuk menjalankan aplikasi, ketik perintah:
php artisan serve
